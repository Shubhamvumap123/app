## 2024-05-23 - [Selection Interface Accessibility]
**Learning:** Using Radix `ToggleGroup` (via shadcn/ui) for selection grids (like time slots) is superior to manual button grids because it provides built-in roving tabindex and arrow key navigation, reducing tab fatigue for keyboard users.
**Action:** Replace manual button selection grids with `ToggleGroup` primitives in future forms (e.g., coach selection, program filters).
